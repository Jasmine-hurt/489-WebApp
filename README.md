# Enhanced Quizlet
Group project for CptS 489

## Project Setup
1. Clone the Repository
```
git clone <repo-url>
cd <project-folder>
```

2. Go to the EnhancedQuizlet folder
```
cd EnhancedQuizlet
```

3. Install packages
```
npm install
```

4. Create .env file in the EnhancedQuizlet folder
*We will add a comment in Canvas of what to add in there

5. Run it
```
nodemon
```
